# ADMESH-Domains site

Static site published to GitHub Pages. Plain HTML + ES modules + Leaflet (CDN). No framework, no npm.

## Build locally

```bash
python scripts/build_site.py
python -m http.server -d site/dist 8000
# open http://localhost:8000/
```

## Layout

```
site/src/        Hand-edited source
  *.html         Pages (one per route)
  styles.css     Theme (CSS custom properties + dark-mode)
  js/            ES module per page + shared helpers
site/dist/       Generated by scripts/build_site.py — gitignored
```

## Deploy

Pushing to `main` with changes under `site/**`, `registry_data/manifest.toml`,
or `admesh_domains/data/manifest.toml` triggers `.github/workflows/pages.yml`.

The workflow runs `python scripts/build_site.py`, uploads `site/dist/` as a
Pages artifact, and deploys via `actions/deploy-pages@v4`.

## Adding a page

1. Create `site/src/<page>.html`.
2. Create `site/src/js/<page>.js` (use `import { renderNav, renderFooter } from "./nav.js"`).
3. Add the page to `PAGES` in `site/src/js/nav.js`.
4. Rebuild (`python scripts/build_site.py`).
